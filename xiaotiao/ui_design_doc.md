# 智慧农研一体化平台：UI/UX 视觉设计白皮书（国奖标准版）

本文件旨在定义“智慧农研”一体化云平台的视觉与交互标准。设计目标是达到全国大学生创新创业大赛（如“大创”、“互联网+”）国奖水准，核心理念为 **“农业传统与前沿科技的数字融合”**。

---

## 1. 核心设计哲学 (Design Philosophy)

*   **高端 SaaS 化**：借鉴成熟的农业科技 SaaS（如极飞科技、大疆农业）的设计语言，强调数据驱动与专业协作。
*   **非“代码堆砌感”**：避免生硬的 AI 模板感，通过自定义组件、极简布局及精致的微动画提升产品独特性。
*   **品牌强关联 (智谱 GLM)**：全站深度集成智谱 AI 视觉元素，建立“AI 助研”的品牌粘性。

## 2. 视觉识别系统 (Visual Identity)

### 2.1 色彩方案
我们摒弃默认配色，采用一套高对比度、呼吸感强的“农研科技”调色盘。

| 区域 | 颜色 (HEX) | 含义与用途 |
| :--- | :--- | :--- |
| **主基调 (成长绿)** | `#10B981` (Emerald 500) | 代表健康作物，用于主操作、成功状态及作物生长数据。 |
| **协作色 (科技蓝)** | `#3B82F6` (Blue 500) | 代表 AI、数据及云服务，用于图标、图表及智谱 GLM 交互。 |
| **基础底色 (晶莹灰)** | `#F8FAFC` (Slate 50) | 全站大面积背景对比色，搭配 Mesh Gradient 使用。 |
| **预警色 (警示橙)** | `#F59E0B` (Amber 500) | 用于病害初发提醒及环境数据异常告警。 |
| **危机色 (危险红)** | `#EF4444` (Red 500) | 用于严重病害爆发、系统故障等高危提示。 |

### 2.2 字体与版面
*   **字体栈**：`Inter`, `-apple-system`, `PingFang SC` (优先考虑系统无衬线字体以保证加载速度与质感)。
*   **字号标准**：13px (备注) / 14px (正文) / 16px (组件文字) / 24px+ (标题)。
*   **行间距**：正文 1.6 - 1.8，确保长时间阅读 AI 生成文档的舒适度。
*   **圆角规范**：全站统一采用 **12px - 16px** 弧度。

## 3. 核心模块美化标准

### 3.1 智谱 GLM 智能助研入口
*   **品牌展示**：AI 助手全程展示为“智谱 GLM 智能助研”，图标采用晶莹剔透的玻璃质感。
*   **Markdown 文档化**：
    *   **排版标准**：严格执行 GitHub 风格的 Markdown 渲染。
    *   **交互细节**：代码块带有一键复制、加粗文字带有渐变背景阴影、表格采用斑马纹且包含农业术语自动加注。
    *   **打字机效果**：回复过程平滑无闪烁，搭配实时生成的波纹动画。

### 3.2 登录与门户体系
*   **SaaS 分栏布局**：禁止传统居中卡片，采用左右 split-screen。
    *   **左侧 (品牌区)**：高清田间作业图 + 半透明 Frosted Glass 遮罩 + 动态 Mesh Gradient 几何背景。
    *   **右侧 (交互区)**：极简的白底表单 + Tailwind 特效输入框。
*   **入场动画**：采用 `Animate.css` 结合 Tailwind 的 `duration-500` 慢入场效果。

### 3.3 AI 检测与实时监控 (YOLO)
*   **扫描特效**：检测时前端增加一条从上至下的蓝色“激光扫描线”动画。
*   **识别卡片**：检测结果不使用弹出框，而是以“悬浮信息卡”形式吸附在画面侧边，卡片包含精美的权重分布图。

## 4. 交互与动效标准 (Interaction UX)

1.  **全局 Mesh Gradient (网格渐变)**：背景设置 3-5 个不同色值的模糊球（Emerald & Blue），随窗口移动有微弱的视差偏移。
2.  **玻璃拟态 (Glassmorphism)**：侧边栏与页头使用 `backdrop-filter: blur(16px)`，半透明度维持在 0.7 - 0.85 之间。
3.  **操作反馈 (Loading Locks)**：所有涉及 API 请求的按钮（如搜索、删除、导入）必须带有 `loading` 动画，防止评委重复点击导致异常。
4.  **Tailwind 优先**：全站布局、间距、阴影统一使用 Tailwind CSS 变量，确保代码的极高标准。

> - 保证全站没有任何“AI 感过重”的廉价图库图。
